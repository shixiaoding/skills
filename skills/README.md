# Skill 索引

本目录中的每个一级子目录都是一个可独立安装的 Skill。

| Skill | 简介 | 文档 |
| --- | --- | --- |
| [research-summary](research-summary) | 收集、整理、维护主题资料库，并按需同步到 Notion | [使用教程](../docs/research-summary/usage.md) |
| [study-examiner](study-examiner) | 一题一答测验已阅读材料，最后统一复盘补弱 | [使用教程](../docs/study-examiner/usage.md) |

不要把教程、文章或一次性调研结果放在本目录；它们应归入 `docs/`。这样安装路径始终保持稳定：`skills/<skill-name>/`。
